# 📊 Data Profiler Project

## 📌 Overview

The **Data Profiler** project is designed to perform exploratory data analysis (EDA) and generate meaningful insights from raw datasets. It helps in understanding data structure, identifying missing values, detecting anomalies, and preparing data for further machine learning tasks.

---

## 🚀 Features

* Data loading and inspection
* Handling missing values
* Data type correction
* Statistical summary generation
* Duplicate value detection
* Outlier detection
* Data visualization (histograms, distributions, correlations)
* Clean dataset preparation for ML models

---

## 🛠️ Technologies Used

* Python 
* Pandas
* NumPy
* Matplotlib
* Seaborn (optional)
* Jupyter Notebook

## ⚙️ Installation & Setup

1. Clone the repository or download files
2. Install required libraries:

```bash
pip install pandas numpy matplotlib seaborn
```

3. Open Jupyter Notebook:

```bash
jupyter notebook
```

4. Run `Data profiler.ipynb`

## 📊 Steps Performed

### 1. Data Loading

* Import dataset using pandas
* Preview data using `.head()`

### 2. Data Cleaning

* Handle missing values
* Remove duplicates
* Fix inconsistent formats

### 3. Data Analysis

* Descriptive statistics
* Correlation analysis
* Distribution plots

### 4. Visualization

* Histogram plots
* Boxplots for outliers
* Heatmaps for correlation
